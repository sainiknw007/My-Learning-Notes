What how and Why

What is home directory ?

/home --> home is system created directory that contains the home directory for almost all users in the Linux Systems.

/home/Michael --> is home directory for user Michael. home directory is unique for user.

another user john will have its own different home directory created..
/home/john

Each user in the Linux system get its own home directory and complete access to it. to save, retrieve or delete data.
other users cannot access your files and folders in your home directory.

Why do we need a home directory ?

Answer: home directory allows users to store their personal data in the forms of files and directories.

Representation of home directory ?

Ans it is represented by ~ tilde symbol
Home directory = ~ tilde symbol
